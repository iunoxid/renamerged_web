Cara Pakai Renamerged

Pilih Mode Proses

Mode Rename Saja

Gunakan jika: Anda hanya ingin mengganti nama file PDF tanpa menggabungkannya.

  • File akan diganti nama sesuai format yang Anda pilih
  • File tetap terpisah berdasarkan IDTKU
  • Cocok untuk arsip atau organizasi file

Mode Rename + Merge

Gunakan jika: Anda ingin mengganti nama dan menggabungkan PDF dengan partner yang sama.

  • File dengan partner sama akan digabung jadi satu
  • Menghemat ruang penyimpanan
  • Cocok untuk laporan bulanan atau tahunan

Langkah-Langkah Penggunaan

1

Persiapkan File

Kumpulkan semua PDF faktur pajak yang ingin diproses ke dalam satu folder, lalu compress menjadi file ZIP.

2

Pilih Mode & Pengaturan

Pilih mode proses yang sesuai kebutuhan Anda:

  • Rename Saja: Konfigurasikan format nama file di panel pengaturan
  • Rename + Merge: Mode ini menggunakan format default
3

Upload File ZIP

Drag & drop file ZIP ke area upload atau klik untuk memilih file. Sistem akan mendeteksi jumlah PDF secara otomatis.

4

Proses & Monitor

Klik tombol "Upload & Proses" dan pantau progress secara real-time di log box. Estimasi waktu akan ditampilkan otomatis.

5

Download Hasil

Setelah proses selesai, akan muncul summary hasil dan link download. File hasil sudah diorganisir sesuai IDTKU.

Cara Mengatur Format

  1. Centang komponen yang ingin disertakan
  2. Drag & drop untuk mengatur urutan komponen
  3. Pilih pemisah antar komponen (-, _, |, dll)
  4. Pilih pengganti karakter "/" jika ada
  5. Lihat preview nama file di bawah pengaturan

Tips & Troubleshooting

Performance Tips

  • File ZIP sebaiknya tidak lebih dari 200MB
  • Pastikan koneksi internet stabil
  • Jangan tutup browser saat proses berjalan

Troubleshooting

  • File tidak terdeteksi: Pastikan file berformat PDF dan dalam folder ZIP
  • Progress terhenti: Refresh halaman dan coba lagi
  • Download gagal: Cek koneksi internet dan coba download ulang

Best Practices

  • Gunakan nama file PDF yang jelas sebelum upload
  • Pastikan PDF tidak corrupt atau password protected
  • Backup file asli sebelum proses

Frequently Asked Questions

Q: Apakah file saya aman?

A: Ya, file akan otomatis dihapus dari server setelah 1 jam. Kami tidak menyimpan data Anda.

Q: Kenapa hanya mendukung ZIP?

A: Saat ini fokus pada format ZIP untuk stabilitas. Support RAR dan 7Z akan ditambahkan di update mendatang.

Q: Berapa maksimal file yang bisa diproses?

A: Maksimal 200MB per ZIP. Jumlah file PDF tidak dibatasi selama total ukuran tidak melebihi limit.

Q: Bagaimana jika ada error saat proses?

A: Error akan ditampilkan di log box. File yang berhasil tetap bisa didownload, yang error akan dilewati.
